Local Storage - Textarea


Referência:
https://www.w3schools.com/jsref/prop_win_localstorage.asp

As propriedades localStorage e sessionStorage permitem salvar pares de chave/valor (key/value) no navegador da WEB.

O objeto localStorage armazena dados sem data de validade. Os dados persistem no navegador, permanecem quando o navegador for fechado e estarão disponíveis por dias, semanas ou anos.

Dica: A propriedade sessionStorage armazena dados para uma sessão (os dados são perdidos quando a guia do navegador é fechada).

No browser veja em Application, Local Storage as variávies registradas (Ctrl + Shift + i)

É possível armazenar estururas de dados tal como JSON em localStorage, convertendo para texto.
